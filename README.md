## 개요

지정한 화면 영역을 주기적으로 캡처한 뒤, 캡처 이미지에서 특정 템플릿 이미지를 탐지하고 결과를 디스코드 채널로 전송하는 자동화 스크립트입니다.
Windows 환경을 기준으로 작성되었습니다.

---

## 주요 기능

### 1. 실행 시 사전 탐지 테스트

* 프로그램 시작 직후 1회 수행
* `captured/captured_test.png` 이미지에서
  `search/search_test.png` 템플릿 탐지 여부 확인
* 결과를 콘솔에 `PASS` 또는 `FAIL`로 출력

### 2. GUI 기반 캡처 영역 지정

* 반투명 전체 화면 UI 제공
* 마우스 드래그로 캡처 영역 지정

### 3. 캡처 → 탐지 → 디스코드 전송

* 캡처 이미지 `captured` 폴더에 저장
* `search` 폴더 내 템플릿 이미지들과 OpenCV 매칭 수행
* 기본 캡처 이미지는 항상 전송
* 템플릿 탐지 성공 시 추가 알림 메시지 전송

### 4. 스케줄 실행

* 짝수 시간 정각(HH:00)에만 실행
* 해당 시점마다 1회 수행

## 설치 및 실행

### 1. 의존성 설치

```bash
pip install Pillow opencv-python numpy requests
```

---

### 2. 디스코드 설정

루트 경로에 `discord.json` 파일 생성 후 아래와 같이 입력

```json
{
  "bot_token": "YOUR_DISCORD_BOT_TOKEN",
  "channel_id": "YOUR_CHANNEL_ID"
}
```

---

### 3. 파일 및 폴더 구성

필수 파일:

* `captured/captured_test.png`
  → 초기 탐지 테스트용 기준 이미지

* `search/search_test.png`
  → 초기 탐지 테스트용 템플릿

추가 템플릿:

* `search` 폴더에 `.png`, `.jpg`, `.jpeg` 파일 추가 가능

---

### 4. 실행

```bash
python remind.py
```

---

## 동작 흐름

1. 프로그램 시작
2. 테스트 이미지 기반 1차 탐지 수행
3. 결과 콘솔 출력 (`PASS` / `FAIL`)
4. `discord.json` 로드 및 설정 검증
5. GUI에서 캡처 영역 선택
6. 선택 즉시 테스트 캡처 실행

   * 이미지 저장
   * 디스코드 전송
   * 템플릿 탐지 및 추가 알림 처리
7. 이후 대기 루프 진입

   * 짝수 시간 정각마다 동일 작업 반복

---

## 참고

* 템플릿 매칭 정확도는 이미지 해상도, 스케일, UI 변화에 영향을 받습니다.
* 캡처 대상 환경이 변경될 경우 템플릿 이미지를 갱신하는 것이 필요합니다.
* Windows DPI 설정이나 화면 배율에 따라 탐지 결과가 달라질 수 있습니다.
